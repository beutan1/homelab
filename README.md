# Background
I struggled a lot with finding a project that I could commit to. Each time that I started something, it felt much more draining and tiring than the last, and I really couldn't care for it. It didn't take long for me to realize that I was doing something wrong.
Before long, I realized that I had to try something else. I decided to use my Thinkpad T480s that I purchased as a Linux machine and repurpose it into a home server since I had interest in IT, Networking, and Cybersecurity. 

Thus, I installed Ubuntu Server on it, instead of having it collect dust on the carpet.

# Uses
Having now had this server for about 1 month, I have been using it to globlly block ads using AdGuard Home, and to stream my personal CD collection/music and downloaded movies from anywhere. This allows me to save money from subscription services like Spotify or Netflix, and having used TailScale, provide free music/movie streaming to my girlfriend and I!

More info can be found in the `/docs` folder, starting with [Initial Setup](</docs/Initial Setup.md>).
